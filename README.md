#### notes
1. 伙伴算法
    内存块按照2的倍数进行划分。
    空闲内存块以链表形式组织。
    用伙伴位图来记录伙伴内存块的空闲情况。
    分配内存时，如果小内存块没有了，就把大内存块分裂成2个小内存块。
    当互为伙伴关系的小内存块都空闲时，合并为大内存块。
